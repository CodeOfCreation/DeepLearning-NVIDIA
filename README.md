# NVIDIA Deep Learning – Learning Repository

This repository contains all the files, notes, exercises, and projects I created while completing the **Getting Started with Deep Learning** course by NVIDIA.

---

## 📚 About the Course

The **Getting Started with Deep Learning** course introduces the fundamentals of deep learning, including neural networks, model training, and practical implementation using modern frameworks and GPU acceleration.

---

## 📁 Repository Structure

```
.
├── notebooks/          # Jupyter notebooks with exercises and demos
├── datasets/           # Sample datasets used in training and experiments
├── models/             # Saved models and checkpoints
├── scripts/            # Python scripts for training and evaluation
├── outputs/            # Results, visualizations, and logs
└── notes/              # Personal notes and key concepts
```

## 🚀 How to Use

1. Clone this repository:

   ```bash
   git clone https://github.com/your-username/nvidia-deeplearning-notes.git
   cd nvidia-deeplearning-notes
   ```

2. Install dependencies:

   ```bash
   pip install -r requirements.txt
   ```

3. Run Jupyter Notebook:

   ```bash
   jupyter notebook
   ```
